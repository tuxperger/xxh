## Единственный источник истины и отсутствие рантайм-Nix

- [x] CHK001 Явно ли зафиксировано, что **канонический конфиг-файл — единственный источник истины** для рантайма? [Completeness, Покрыто, Spec §FR-041 / Принцип XI]
- [x] CHK002 Задано ли, что модуль **лишь генерирует** этот файл, а не является альтернативным источником конфигурации? [Clarity, Покрыто, Spec §FR-042 / Contract C-CM1]
- [x] CHK003 Определено ли требование **отсутствия рантайм-зависимости от Nix**? [Completeness, Покрыто, Spec §FR-042 / Contract C-CM2]
- [x] CHK004 **Измеримо** ли, что сгенерированный конфиг читается без Nix в рантайме? [Measurability, Покрыто, Spec §SC-014]
- [ ] CHK005 Непротиворечив ли этот принцип с существующим правилом precedence (флаг CLI > конфиг), т.е. рантайм читает только канонический файл + флаги? [Consistency, Двусмысленно, §FR-041 упоминает флаги, но связь с §FR-024 precedence явно не сшита]
